一个做安全开发的佛系少年的自留地

终章

2021-01-16

随着耳边响起一阵发动机的咆哮声，我踏上了公历年前的最后一次航行，而这次的目的地恰是年初第一次航行的始发地。兜兜转转度过了这魔幻的一年，最后以这种方式完成了一次产品经理口中经常提到的“闭环”。在 1613088000 这个时间戳不可阻挡的到来之前，我想为 2020 做个后记，记录下一个以我为中心的“人类观察报告”。

以色列计划

在疫情到来之前，我本以为本年将是十分充实的一年。因为各种神奇的因素叠加在一起，我有机会以与会者的身份公费参观学习以色列的某网络安全大会。机票、签证都一切顺利，手续之类的有贴心的市场部小姐姐帮忙处理，我除了看了一下大会中的议题这个正事外，更多的都是做攻略去了。比如恶补了一下以色列的相关历史，对犹太人曲折建国的过程有大概的了解，不至于去了一些名胜却像个文盲一样；也随手调查了一下本土的特色美食，以期能体验到“舌尖上的以色列”； 甚至还做了类似“以色列生存指南”的研究，以防有什么突发情况可以+1秒。

准备的差不多大家就回家过年了，实际上放假前夕疫情就有零星疫情了，只是开始都没有太在意，我记得那时候还随手买了一盒 N95 防雾霾口罩用来对抗北京冬天恶劣的生存环境，没想到的是，这盒口罩成了我手中升值最快的“理财产品”。春节没过完，手机就接连不断的弹出一些骇人的消息，我有点担心这行程会不会被取消了。随后的事情大家都经历过了，疫情以燎原之势席卷了整个荆州，国外也出现了一些案例，国家开始动用一些强制措施减少人员流动降低传染风险。为了防止封城后去不了北京，我大年初二就从家出发去了坐上了前往北京的高铁。在高铁上有个小孩一直在哭闹咳嗽，出于对未知的恐惧，我离开座位在车厢连接处站了4个小时，因为那里没有人。(图为 2 月 2 日正月初八下起了大雪，此时我已经在北京了 )

后来不出所料的，在起飞的前一天航班被取消了，虽然有些可惜但取消绝对是正确的选择，毕竟命才是最重要的。要问北京什么时候不堵车，春节这段时间绝对算得上头牌。偌大的北京城街道上空无一人，在疫情的衬托下恍如末日一般将大家禁锢在了家里。比较幸运的是，我的工作没有受太大影响，除了远程办公外没有别的变化，感谢阿里爸爸这座大山。再后来疫情逐渐变得可控，一切也渐渐恢复了往日的吵闹。对于这段经历，我保留了一笔遗产给自己。出国前我曾兑换了一些美元，计划取消后我也没有兑换回来，而是作为一段记忆留下了，并通过结婚红包、新年压岁钱等方式赠给了一些朋友和亲人，大家也觉得蛮有趣的。我给自己保留了几张，希望将来有机会出国再派上用场。

旅行

江山多娇，不为英雄，而为过尽千帆，揽阅万物。

天津

这是我第一次说走就走的旅行。周五下午买票，晚上下班出发，路上定了酒店，摆脱计划与规则的感觉真棒，一切随心所欲，所见所闻都是意料之外。酒店就定在了车站附近，我落脚后到海河边上逛了逛，随手拍了几张照。

第二天我坐车去了滨海新区，找了个靠海的酒店，以为能像在青岛一样吹吹海风玩一下，可惜被自己打脸。这边没有海滩，周边建筑也很不完善，有种小渔村的感觉，我住的酒店和附近低矮的平房有些格格不入。不过我还是很开心，毕竟新鲜的事物总是令人兴奋。

天津有着众多极具特色的建筑，从中式的亭台到西式的洋房，从古典洋房到现代高楼大厦，有点目不暇接。可惜那天时间紧凑，我没怎么逛就坐上回去的车了，这是此行小小的遗憾。

济南

毫不掩盖的说，济南不是我的目的地，它只是我去西安的中转站。要不是我买的随心飞有从北京直达西安的航线，我恐怕会错过这段旅程。


大明湖名不虚传，一片片荷花池令人很是兴奋。这里的荷花像是雨林中的树木一样拥簇着，郁郁葱葱，极具生命力，让人忍不住驻足观赏。我去的时候荷花还未开也可能是开过了，即便这样，我依然对这景象印象深刻。大明湖铁公祠上有一副对联，恰到好处的描绘了我对济南的感觉：四面荷花三面柳，一城山色半城湖 。

西安

素闻长安有 N 朝古都之称，一直想去看看，告别了大明湖后，我踏上了前往咸阳机场的飞机。这个机场距离市区的距离有点离谱，感觉就不在一个城市，兜兜转转了近两个小时才落脚。在西安的那两天时不时就下起了毛毛雨，一个北方内陆城市竟有了杭州烟雨小城的感觉。我对这种雨不讨厌，当然也谈不上喜欢，因为相比之下我更喜欢万里无云的湛蓝晴空。

故里

19年国庆的时候，我不知哪来的勇气在国庆去了趟西湖，那时我才体会到什么是真正的人海和人山，往后我便学该乖了。今年国庆同时也是中秋节，我想也没想直接就回老家了。这8天生活感觉像是预知了我的退休生活，和我的发小们过上了“一曲高歌一樽酒，众人共钓一江秋”的闲适生活。（这句诗后半句是我瞎编的）


后面去了一趟茶山，这必须得感谢下我的小表弟，可以说是我表弟费九牛二虎之力拉着我去的，去了之后发现这边的景观和设施属实有点夸张。景区依山而建，岩、泉、花、果、寺在其中随处可见，甚至还有那种比较朋克的岩洞旅馆和餐厅，不过我没去体验。我没想到在一个地级市有这么庞大的旅游投入，后来查了下竟然还是个 4A 级风景区。值得一提的是，这个山上种了一片板栗树，去的时候恰是栗子成熟的时候，地上掉了一地，我们好心的帮园区采集了一部分，走的时候每个人的口袋都硕果累累，此行无论精神还是物质上都称得上血赚。

三亚

由于我买的随心飞没有从北京直达三亚的航班，而各种转机下来时间成本太高，索性放弃了套餐自费买了时间比较合适的机票，不过好在是旅游淡季，这一趟下来也只花了两千多，考虑到三亚的消费水平这应该算是穷游了。我和同事都是山东人，之前也从没去过这么南的地方，心里还是有很强的期待的。我俩很粗糙的做了下攻略，把酒店定在了相对大众化的大东海附近，找了一个出门是海滩的酒店，然后就开始了放浪不羁的度假生活。

小伙伴洋仔是个排球迷，此行对沙滩排球一直念念不忘，就像寻宝一样，到处打听哪里有沙滩排球，后来还真被他找到了，和一群业余而专业的选手们同台竞技了一下之后，便感叹自己太胖了跳不起来导致发挥失常，我趁机讽刺了一下他那富有弹性的肚皮，不过他似乎对我的嘲讽已经自动免疫，说来这也有我一半的责任，毕竟当时一拍而合办的健身卡都不约而同的过期了。


这时候肯定会有小可爱问我去干啥了，我只管躺平了。“人与人的欢乐并不相通，我只是觉得他们吵闹”在看我来吹吹海风，听听海浪，晒晒太阳，游个泳再晒晒太阳，这就是无比浪漫的事情了。在海边真的可以让人放下一下烦恼，沉醉在一声声波涛拍岸的声音中，静静聆听，这世界便只剩两种声音，一是在浪花之上，以高傲的姿态不断向高点冲刺的声音；一是暗流之下，默默支撑期待能成为下一朵后浪的声音。这两种声音此起彼伏，延绵不绝，相辅相成而又互相扼喉，这样简单的区分像极了计算机中的 0 和 1，我喜欢这般原始而简单的生活，这没有人情世故和新仇旧怨的生活，这样的生活在现实不存在，但存在于动漫中。

动漫

时间都去哪了，都被二次元的黑洞吞噬了。不过我只能称为半二次元患者，或者叫动漫爱好者更好一些。2020 年其实我没看几部，但感觉都蛮有意思的，不妨也记录下。（排名不分先后）

Re0: 从零开始的异世界生活

《Re0》的第二部是伴随着第一部的“新编集”版一起上线的，第一季剧情记不太清了我就先补了一下。第一季给我印象最深的除了你们的老婆雷姆拉姆就是那个魔女教大罪司教怠惰，我感觉这个鬼就是周星驰无厘头的集大成者，抱歉虽然它是恶人但它真的太可爱了。

第二部中我和无数人一样也投敌了，就是那个艾姬多娜，有一集对主角长达3分钟数千字的告白，给我看的目瞪口呆，当场拿出小本本记下来，等我有了女朋友我也念给她听。

辉夜大小姐想让我告白

年度最佳恋爱喜剧！我看的时候全程姨母笑，高手过招，招招沙雕。俩人都是天才，但在恋爱这件事上集体降智，本来平A就能赢的事硬生生是走位了整整一季，要是没有那么多骚操作，俩人早就在一起了。这部动漫深刻的说明了两个道理：

  • 开始的分分钟都妙不可言
  • 恋爱使人变婴

鬼灭之刃

《鬼灭之刃》本身的剧情其实是老套路，无非是一个热血少年打怪升级拯救世界的故事。但他的动画制作实在太优秀了，在无敌配音的加持下打造了一部人物刻画细腻鲜明、性格栩栩如生的罕见佳作，打斗极具代入感，看的我热血沸腾，血压都不正常了，上一次有这种感觉的还是《巨人》系列。我找了第一部里个人最喜欢的图片做了电脑壁纸，就是决战小蜘蛛男时最后一击的场景，窃以为这是第一部的巅峰时刻。（期待《无限列车篇》能在国内上映...)

雾山五行

国漫之光！雾山五行只有三集，而且这3集的大部分工作都是由一个叫林魂的人独自完成的，这是真的大佬。这种水墨神话武侠作真的太和我胃口了，完美的诠释了什么叫小而美，看完3集后大呼不过瘾，希望有生之年能看到这个动漫的续集，如果能保持前三集的水平，这绝对是划时代的杰作。

异度侵入 ID: INVADED

这是一部有种《盗梦空间》感觉的悬疑侦探类动漫，之前没有看过这种类型的，新鲜感爆棚。动漫设定上有个令人很矛盾的地方，有资格进入“井”去做正义侦探的人都是前杀人犯、前自杀者等看起来性格有缺陷的一类人，我认为这是这部动漫的立高点，OP 中开头的两句就直击了这一主题。

今夜依然居高而问 「你也是伪善者吗」
正义是什么 造一个便是

工作

没想到洋洋洒洒写了这么多，现在也是时候给工作写点总结了。我并不打算去罗列一些和自己相关的数字变化，那是给领导汇报用的，而是希望能看到自己一些内在的成长。过去一年我的主旋律依然是 xray，很高兴和大家一起做了有趣的事。得益于项目上的要求和大佬们的帮助，一句话概括下这一年的工作状态就是：解决未知问题的能力和学习新事物的能力里进一步加强，更诗意一些的，探索世界的能力更强了。

我是一个喜欢做梦的人。刚毕业那会，踌躇满志，看着这个新鲜和神奇的世界，幻想着能用自己的双手给这个世界画上属于自己的一笔。两年间，这个想法竟以某种形式真的存在了，很高兴，这两年我拼尽全力。

接下来，我也有了新的念想。我相信这世界的基调依然是美好，我想在年轻时去探索这个世界众多美好的不确定性，并将其烙印到未来确定的人生轨迹中。也许这有些抽象，但这不重要，重要的是我已经准备好启航，也做好了迎接残酷现实的准备。

以上，是我离职申请邮件的部分内容。是的，我离开长亭了。在我给这段故事画上句号的同时也开启了新的征途，不要问我去哪，我也不清楚，因为那尚且虚无缥缈。回顾两年多工作欢声笑语始终陪伴，能结识这么多 nice 的朋友感觉很幸运，衷心感谢大家的关注和照顾。过去的只能作为记忆留念了，相比之下，未来更值得期待。愿大家不惧挫折，步步为营，离自己理想中的生活越来越近。江湖路远，大家后会有期。
